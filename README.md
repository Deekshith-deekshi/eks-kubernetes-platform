# Scalable Kubernetes Platform on AWS (EKS)

This project demonstrates a production-ready Kubernetes platform deployed on **Amazon EKS**.

## 🚀 Features
- Deployment of microservices on EKS
- Helm charts for application packaging
- Horizontal Pod Autoscaler (HPA) for dynamic scaling
- Monitoring stack with Prometheus & Grafana
- AWS Load Balancer integration for traffic management
- Infrastructure provisioned using Terraform

## 🛠️ Tools & Technologies
- AWS (EKS, IAM, VPC, EC2, Load Balancer)
- Kubernetes & Helm
- Terraform
- Prometheus & Grafana
- CI/CD with GitHub Actions / Jenkins

## 📈 Outcome
A scalable, monitored, and automated Kubernetes platform suitable for production workloads.
